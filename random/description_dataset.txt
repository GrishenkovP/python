id - идентификатор абонента
name - имя абонента
state - штат
date_con - дата подключения
tariff_plan_con - название тарифа, на который был подключен абонент
date_change_tariff_plan - дата смены тарифного плана (если она проводилась)
tariff_plan_change - наименование тарифа после смены (если первоначальный тариф не менялся, то поле должно быть пустым)
date_discon - дата ухода абонента (если поле незаполнено, то абонент еще пользуется сервисом)
